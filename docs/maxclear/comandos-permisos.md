# Comandos y Permisos <Badge type="info" text="MaxClear" />

::: info Sintaxis de Comandos
Recuerda la estructura de los comandos antes de usarlos:
* `[ ]` **Argumento Obligatorio**
* `< >` **Argumento Opcional**
:::

## Lista de Comandos

| Comando | Alias | Uso | Permiso |
| :--- | :--- | :--- | :--- |
| `/maxclear` | `/mc` | `/maxclear` | `maxclear.admin` |
| `/maxclear` | - | `/maxclear help` | `maxclear.admin` |
| `/maxclear` | - | `/maxclear reload` | `maxclear.admin` |
| `/maxclear` | - | `/maxclear clear` | `maxclear.admin` |
| `/maxclear` | - | `/maxclear get <author/version>`| `maxclear.admin` |

::: warning Acceso Total
El permiso `maxclear.admin` da acceso a todos los comandos de optimización y limpieza del plugin.
:::
