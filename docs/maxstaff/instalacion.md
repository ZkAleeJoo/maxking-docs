# Instalación

MaxStaff Premium está diseñado para funcionar de manera óptima en el ecosistema de Paper/Purpur, requiriendo la API de la versión 1.19 o superior.

## 📥 Pasos de Instalación

Sigue estos pasos para instalar el plugin correctamente:

1. Compila o descarga el archivo `.jar` de MaxStaff Premium.
2. Coloca el archivo en la carpeta `/plugins` de tu servidor.
3. Reinicia el servidor por completo para generar los archivos de configuración.
4. Revisa y ajusta los archivos `config.yml`, `discord.yml` y tu idioma preferido (`messages_es.yml` o `messages_en.yml`).
5. *(Opcional)* Instala PlaceholderAPI si deseas hacer uso de las variables interactivas del plugin.

> **Métricas:** MaxStaff incluye las métricas de bStats por defecto y cuenta con un sistema de verificación de actualizaciones al iniciar, el cual puede ser configurado.
