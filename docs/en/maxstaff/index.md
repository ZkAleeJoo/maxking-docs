# INSTALLATION AND REQUIREMENTS

> In this section, you will learn how to install **MaxStaff** on your server and perform the basic initial configuration to ensure the plugin works properly. As always, you can join our Discord server: [https://discord.gg/Vr46JHm2kd](https://discord.gg/Vr46JHm2kd)

## 1. Server Requirements

> Before proceeding, make sure your environment meets the following requirements:

* **Minecraft Version:** 1.21.x - 26.1.2
* **Java:** Version 21 or higher.
* **Server Software:** Paper, Purpur, Pufferfish, or any fork compatible with the Paper API.

## 2. Optional Dependencies

> Although the plugin works on its own, we recommend installing:

* [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/): Required to display punishment statistics and staff states in other plugins (Scoreboards, TabLists, etc.). This is recommended if you want more insight into what is happening on your server. You can find **MaxStaff** placeholders in the [MaxStaff Placeholders](/maxstaff/placeholderapi) section.

## 3. Installation Guide

* Download the latest version **.jar** file of **MaxStaff**.
* Upload the file to your server’s `/plugins/` folder.
* Restart the server (or use a plugin manager) to generate the configuration files.
* Done! The plugin will now be active with the default configuration.

## 4. Initial Configuration

> By default, the plugin is generated in English. Follow these steps to change it to Spanish:

1. Go to the `/plugins/MaxStaff/` folder.
2. Open the `config.yml` file.
3. Locate the `language` option inside the `general` section.
4. Change `"en"` to `"es"`.
