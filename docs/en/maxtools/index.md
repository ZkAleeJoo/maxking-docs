# INSTALLATION AND REQUIREMENTS
> In this section you will learn how to install **MaxTools** on your server and perform the basic initial configuration so that the plugin works properly, as always you have the discord server https://discord.gg/Vr46JHm2kd

## 1. Server Requirements
> Before continuing, make sure your environment meets the following requirements:
- **Minecraft Version:** 1.21.x - 26.1.2
- **Java:** Version 21 or higher
- **Server Software:** Paper, Purpur, Puperfish or any fork compatible with the Paper API.
 
## 2. Installation Guide
- Download the **.jar** file of the latest version of **MaxTools**.
- Upload the file to the `/plugins/` folder of your Minecraft server.
- Restart the server (or use a plugin loader) to generate the configuration files.
- Ready! The plugin will now be active with the default configuration.

## 3. Initial Configuration
> By default, the plugin is generated in English. Follow these steps to change it to another language:
1. Open the game and run the command **/met lang**
2. Choose your preferred language.
3. Ready! The plugin will now be active with the default configuration.
