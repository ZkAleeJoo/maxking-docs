# INSTALLATION AND REQUIREMENTS

> In this section, you will learn how to install **MaxGraves** on your server and perform the basic initial configuration to ensure the plugin works properly. As always, you can join our Discord server: [https://discord.gg/Vr46JHm2kd](https://discord.gg/Vr46JHm2kd)

## 1. Server Requirements

> Before proceeding, make sure your environment meets the following requirements:

* **Minecraft Version:** 1.21.x (Specifically tested on 1.21.11).
* **Java:** Version 21.
* **Server Software:** Spigot, Paper, or any fork compatible with the Spigot API.

## 2. Installation Guide

* Download the latest version **.jar** file of **MaxGraves**.
* Upload the file to your server’s `/plugins/` folder.
* Restart the server (or use a plugin manager) to generate the configuration files.
* Done! The plugin will now be active with the default configuration.

## 3. Language Configuration

> By default, the plugin is generated in English. Follow these steps to change it to Spanish:

1. Go to the `/plugins/MaxGraves/` folder.
2. Open the `config.yml` file.
3. Locate the `language` option inside the `general` section.
4. Change `"en"` to `"es"`.
