# INSTALACIÓN Y REQUISITOS
> En esta sección aprenderás cómo instalar **MaxGraves** en tu servidor y realizar la configuración inicial básica para que el plugin funcione de buena manera, como siempre tienes el servidor de discord https://discord.gg/Vr46JHm2kd

## 1. Requisitos del servidor
> Antes de continuar, asegúrate de que tu entorno cumpla con los siguientes requisitos:
- **Versión de Minecraft:** 1.21.x (Probado específicamente en 1.21.11).
- **Java:** Versión 21.
- **Software del servidor:** Spigot, Paper o cualquier fork compatible con la API de Spigot.

## 2. Guía de instalación
- Descarga el archivo **.jar** de la última versión de **MaxGraves**.
- Sube el archivo a la carpeta `/plugins/` de tu servidor de Minecraft.
- Reinicia el servidor (o usa un cargador de plugins) para generar los archivos de configuración.
- ¡Listo! El plugin ahora estará activo con la configuración por defecto.

## 3. Cambio de idioma
> Por defecto, el plugin se genera en inglés. Sigue estos pasos para cambiarlo a español:
1. Ve a la carpeta `/plugins/MaxGraves/`.
2. Abre el archivo `config.yml`.
3. Busca la opción `language` dentro de la sección `general`. 
4. Cambia `"en"` por `"es"`.
